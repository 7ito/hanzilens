# HanziLens

HanziLens helps you read Chinese text on the web. Select Chinese text or capture a screenshot region, then HanziLens breaks sentences into words with pinyin, definitions, and dictionary details.

## Browser extensions

* [Chrome extension](https://chromewebstore.google.com/detail/hanzilens/ikgaoimiiadkcalpiccmalabkhpebfig)
* [Firefox extension](https://addons.mozilla.org/en-US/firefox/addon/hanzilens/)
